# Job Scheduler

A responsive Job Scheduler web application built with JavaScript. It allows users to create, view, update, and delete scheduled jobs through a simple user interface. The project demonstrates JavaScript fundamentals, CRUD operations, local data handling, and responsive UI design.

## Features
- Create new scheduled jobs
- View all scheduled jobs
- Edit job details
- Delete jobs
- Set job title, description, date, and time
- Track job status
- Responsive and user-friendly interface

## Technologies Used
- JavaScript
- HTML
- CSS
- Tailwind CSS

## Concepts Used
- CRUD Operations
- DOM Manipulation
- Event Handling
- Local Storage
- Form Validation
- Responsive UI Design

## How to Run
1. Clone this repository.
2. Open the project folder.
3. Open the `index.html` file in a web browser.
